简单的说下应用结构

``` vue
<html>
<head>
    <!-- 1.引入资源 -->
    <script src='./vue@2.6.10.js'></script>
</head>
<body>
    <!-- 2.定义根DOM元素 -->
    <div id="app"></div>
</body>
<script>
    //3.实例化Vue对象
    var vm = new Vue({
        el: '#app'
    })
</script>
</html>
```

### mustache模板语法
示例代码在data中定义了一个msg，想要在body中显示，使用{{msg}}的形式在网页中显示。
